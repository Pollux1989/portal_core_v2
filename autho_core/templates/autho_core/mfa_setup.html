Protege tu cuenta con un código adicional

La autenticación de dos factores (2FA) añade una capa extra de seguridad a tu cuenta. Además de tu contraseña, necesitarás un código generado por una aplicación de autenticación.

Pasos para configurar 2FA:
  1. Descarga una aplicación de autenticación (Google Authenticator, Authy, etc.)
  2. Escanea el código QR o introduce el código manualmente
  3. Introduce el código de 6 dígitos generado por la aplicación
  4. ¡Listo! Tu cuenta estará protegida con 2FA
